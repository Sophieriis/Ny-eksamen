Grundlæggende Web

I grundlæggende web skulle vi påbegynde vores rejse med kodning i HTML og CSS. Alt fra design af brugergrænseflader til hvordan man benytter sig af Visual Studio Code. Det var dette tema som skabte grundforståelse for hvordan man koder og opbygger en hjemmeside, fra bunden.

Grundlæggende UX

I grundlæggende UX lærte vi at bruge Adobe XD på en helt ny måde, ved at skabe et design som skulle fungere responsivt. Layout og design var elementært, samt brugerens rejse gennem prototypen. Vi testede designs og fandt dermed ud af hvad som skulle ændres og det som fungerede godt.
